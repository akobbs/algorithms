# Problem

Construct a complete binary tree from given array in level order fashion.

### Example 1:

- `Input`: [1, 2, 3, 4, 5, 6]
- `Output`: root of the following tree

```
                  1
                 / \
                2   3
               / \ /
              4  5 6
```

### Example 2:

- `Input`: [1, 2, 3, 4, 5, 6, 7, 8, 9}
- `Output`: root of the following tree

```
                   1
                  / \
                 2   3
                / \ / \
               4  5 6  7
              / \
             8  9
```
